You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- Multiplies two numbers together (format multiply?num1=value&num2=value
  • /github -- List all public repos from the provided user (format github?users/USERNAME/repos)
  • /greaterorless -- Compares two numbers and returns the larger one. exactURL( 3.143.22.214:9000/greaterorless?num1=15324&num2=113 ) (format greaterorless?num1=value&num2=value
  • /wordshuffle -- takes any string and shuffles it a specified amount of time, printing each shuffled string. exactURL (3.143.22.214:9000/wordshuffle?word=photosynthesis&num=10) (format wordshuffle?word=string&num=value
